# TCP Chat Server

A command-line TCP chat server written in C for Linux.

## Features

### Completed

- Protocol module
- Message formatting
- Socket send/receive wrapper
- Partial `send()` handling

### In Progress

- Blocking TCP server
- Protocol testing

### Planned

- Multi-client support
- Broadcast messages
- POSIX threads
- `/list` and `/quit`
- Docker
- GitHub Actions

## Tech Stack

- C
- Linux / WSL2
- POSIX Socket API
- POSIX Threads
- Git
- Docker
